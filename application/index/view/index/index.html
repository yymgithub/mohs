服 务

开源组件依赖扫描

快速扫描出Maven工程依赖的所有开源组件，组件之间的依赖关系。可一键生成组件依赖报告。

开源组件漏洞扫描

快速扫描出Maven工程使用的开源组件的漏洞情况，是否有安全漏洞，漏洞详情以及漏洞级别高低，针对每个漏洞，推荐漏洞解决方案和替代组件。

开源组件License扫描

快速扫描出Maven工程使用的开源组件依赖的所有License的详情、版本、合规性、兼容性。

开源组件热度扫描

快速扫描出Maven工程使用的开源组件当前版本的热度，以及该组件目前所有可使用版本的热度排名情况。

产品

Web站点上传Maven工程进行扫描 && 作为Eclipse插件对Maven工程扫描

我们的产品目前有Web站点上传Maven工程进行健康扫描、作为Eclipse插件对使用Eclipse开发的Maven工程进行健康扫描两种形式。Web站点支持Maven工程开源组件依赖扫描、漏洞扫描、License扫描、热度扫描，可一键生成依赖报告、漏洞报告、License报告、热度报告。针对每一个漏洞，我们可以推荐漏洞解决方案或可替代开源组件。Eclipse插件除了具有web站点所支持的所有功能以外，还可针对工程每个开源组件单独进行漏洞情况、License、热度情况查询，此外，我们会对所有最新导入的开源组件进行自动健康扫描。同时，我们会给出目前所使用开源组件漏洞等级从高到底排序，以便用户及时把握目前急需解决的安全漏洞问题（定期更新并提醒用户查看）。
